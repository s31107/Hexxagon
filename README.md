<h3>Requirements:</h3>
cmake at least 3.25

<h3>How to run (example):</h3>
<h4>Prepare app:</h4>
cd <project path>/Hexxagon
mkdir build
cd build
<h4>Build app:</h4>
cmake ..
make
<h4>Run app:</h4>
./Hexxagon

<h3>Tutorial:</h3>
You can see game tutorial in Hexxagon.mp4
